### 开发

```bash
pnpm i # 如果你没装过 pnpm, 可以先运行: npm install -g pnpm
```

```bash
pnpm dev
```

### 构建

构建该应用只需要执行以下命令

```bash
pnpm build
```

然后你会看到用于发布的 `dist` 文件夹被生成。

### 提交模式

```bash
feat: 新功能
fix: 修复 bug
docs: 文档更新
style: 代码格式调整（不影响逻辑）
refactor: 重构
test: 添加或修改测试
chore: 构建过程或辅助工具的变动
```
